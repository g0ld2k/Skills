# Summarize Skill

## Description
Produces a concise summary of a given piece of text, document, or conversation.

## Usage
Provide text to summarize, along with an optional target length or format.

## Input
- `text` (string): The content to summarize
- `format` (optional, string): `bullet-points` | `paragraph` (default: `paragraph`)
- `max_sentences` (optional, number): Maximum number of sentences in the summary

## Output
A clear, concise summary of the provided content.

## Example Prompt
> Summarize the following article in 3 bullet points:
> <article text here>
